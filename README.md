# 🏦 Банковская система на Python (Advanced OOP)

Учебный проект, демонстрирующий глубокое понимание принципов объектно-ориентированного программирования (ООП) и архитектуры кода.

## 🚀 Основные возможности:
- **Иерархия счетов:** Базовый класс `BankAccount` и наследники: `KidsAccount` (с контролем лимитов) и `GoldAccount` (с автоматическими бонусами).
- **Безопасные транзакции:** Метод `transfer` с логикой отката (rollback) при ошибке зачисления.
- **Логирование:** Полная история операций для каждого аккаунта.

## 🛠 Технические особенности (Pro Level):
- **Инкапсуляция данных:** Использование приватных атрибутов (`__balance`) для защиты критических данных.
- **Геттеры и Сеттеры:** Применение декораторов `@property` и `@balance.setter` для контролируемого доступа к балансу.
- **Полиморфизм:** Переопределение методов родителя для изменения логики в зависимости от типа счета.
- **Делегирование:** Использование `super()` для расширения функционала без дублирования кода.

## 📂 Как запустить:
1. Клонируйте репозиторий.
2. Запустите `BankSystem_OOP.py`.
3. В конце файла представлены готовые сценарии тестирования.
